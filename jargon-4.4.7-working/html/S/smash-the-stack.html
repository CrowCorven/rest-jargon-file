smash the stack `Prev `__  S  `Next `__ -------------- **smash the stack**: n. [C programming] To corrupt the execution stack by writing past the end of a local array or other data structure. Code that smashes the stack can cause a return from the routine to jump to a random address, resulting in some of the most insidious data-dependent bugs known to mankind.
