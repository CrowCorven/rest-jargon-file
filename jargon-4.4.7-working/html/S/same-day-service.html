same-day service `Prev `__  S  `Next `__ -------------- **same-day service**: n. Ironic term used to describe long response time, particularly with respect to `*MS-DOS* <../M/MS-DOS.html>`__ and Windows system calls (which ought to require only a tiny fraction of a second to execute).
