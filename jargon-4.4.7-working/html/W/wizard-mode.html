wizard mode `Prev `__  W  `Next `__ -------------- **wizard mode**: n. [from `*rogue* <../R/rogue.html>`__] A special access mode of a program or system, usually passworded, that permits some users godlike privileges. Generally not used for operating systems themselves (root mode or wheel mode would be used instead).
