bit bashing `Prev `__  B  `Next `__ -------------- **bit bashing**: n. (alt.: bit diddling or `*bit twiddling* `__) Term used to describe any of several kinds of low-level programming characterized by manipulation of `*bit* `__, `*flag* <../F/flag.html>`__, `*nybble* <../N/nybble.html>`__, and other smaller-than-character-sized pieces of data; these include low-level device control, encryption algorithms, checksum and error-correcting codes, hash functions, some flavors of graphics programming (see `*bitblt* `__), and assembler/compiler code generation.
